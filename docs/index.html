GitHub Pages Ready

New Athlete Badminton School

หน้า GitHub Pages สำหรับ public site ของโปรเจกต์นี้ ใช้สำหรับแสดงข้อมูลโรงเรียน, tech stack, และสถานะความพร้อมของระบบ ก่อนเชื่อมต่อกับ deployment หลักของแอปเต็มรูปแบบ

ติดต่อผ่าน Facebook คุยผ่าน LINE

ภาพรวมโปรเจกต์

ระบบนี้พัฒนาด้วย Next.js 14, TypeScript และ Supabase เพื่อรองรับการจัดการนักเรียน, การจองเรียน, ระบบโค้ช, ฝั่งแอดมิน, การชำระเงิน และการติดตามพัฒนาการ

Frontend

Next.js 14

ใช้ App Router, React 18, TypeScript และ TailwindCSS สำหรับ UI หลักของระบบ

Backend

Supabase

ใช้ Auth, PostgreSQL, Storage และ RLS สำหรับ business logic และข้อมูลจริง

Deployment

GitHub Pages + Server Host

GitHub Pages ใช้สำหรับ public/static page ส่วนระบบเต็มยังต้องอยู่บนโฮสต์ที่รองรับ Next.js server features

ฟีเจอร์หลักของระบบ

สำหรับผู้ใช้

  • สมัครสมาชิกและเข้าสู่ระบบ
  • จองเรียนและดูประวัติการเรียน
  • ติดตามพัฒนาการและอันดับ

สำหรับโค้ช

  • เช็คชื่อและเช็คอิน
  • บันทึก Level และโปรแกรมสอน
  • ติดตามชั่วโมงงาน

สำหรับแอดมิน

  • จัดการผู้ใช้ โค้ช และสาขา
  • จัดการการเงิน คูปอง และการจอง
  • ตรวจสลิปและตั้งค่าระบบ

ข้อจำกัดของ GitHub Pages

GitHub Pages เป็น static hosting จึงยังไม่สามารถรัน middleware, API routes, server-side authentication callback หรือ logic ฝั่งเซิร์ฟเวอร์ของ Next.js แอปนี้ได้โดยตรง

  • หน้า public/static แสดงบน GitHub Pages ได้
  • ระบบ login, dashboard, admin, coach และ API ยังควรรันบน Vercel หรือโฮสต์ที่รองรับ Next.js
  • หน้านี้จึงถูกเตรียมไว้เป็น public entry page ของ repository
